# CriptoHub

Plataforma educacional sobre criptomoedas com conteúdo essencial, documentação e colaboração comunitária.

## Estrutura do Projeto

- `/frontend` - Next.js + Tailwind CSS + MDX
- `/backend` - Go (Gin) API REST

## Tecnologias

**Frontend:**
- Next.js 14 (App Router)
- Tailwind CSS
- MDX para conteúdo

**Backend:**
- Go 1.21+
- Gin Framework

## Como Rodar

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
go run main.go
```
